# Módulo 01: Autenticación y Autorización

Este módulo contiene toda la documentación relacionada con el diseño, arquitectura y implementación del sistema de gestión de usuarios, roles y permisos (RBAC) de Aurora Nova.

## Alcance

- Creación de usuarios y gestión de ciclo de vida.
- Autenticación mediante credenciales.
- Gestión de sesiones a través de tokens (JWT).
- Creación y gestión de Roles.
- Creación y asignación de Permisos.
- Diseño de la base de datos para soportar estas funcionalidades.

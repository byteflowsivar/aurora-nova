# Documentación de la Base de Datos

Esta carpeta contiene la documentación técnica detallada sobre el diseño de la base de datos PostgreSQL para el proyecto Aurora Nova.

## Contenido

- **[01_Schema_SQL.md](./01_Schema_SQL.md)**: Scripts SQL `CREATE TABLE` que definen la estructura exacta de cada tabla, incluyendo tipos de datos, constraints y relaciones.

- **[02_Indices_y_Rendimiento.md](./02_Indices_y_Rendimiento.md)**: Estrategia de indexación para optimizar las consultas y cumplir con los requerimientos de rendimiento.

- **[03_Estrategia_de_Seeding.md](./03_Estrategia_de_Seeding.md)**: Plan para la carga de datos iniciales (sembrado) necesarios para el arranque del sistema.

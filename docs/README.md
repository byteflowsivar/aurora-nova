# Documentación del Proyecto: Aurora Nova

Bienvenido a la documentación centralizada del proyecto Aurora Nova. Esta carpeta (`/docs`) sirve como la única fuente de verdad para todos los aspectos conceptuales y de diseño del proyecto.

## Estructura de la Documentación

La documentación está organizada por **módulos funcionales** para facilitar la escalabilidad y la consulta.

- **[01_modules/](./01_modules/)**: Contiene la documentación detallada de cada módulo de la aplicación.
    - **[01_auth_and_authz/](./01_modules/01_auth_and_authz/)**: Módulo de Autenticación y Autorización.

## Cómo Contribuir

- Para documentar un nuevo módulo, crea una nueva carpeta dentro de `01_modules/`.
- Dentro de la carpeta de cada módulo, sigue la estructura de subcarpetas (requirements, architecture, etc.) según sea necesario.
- Siempre que se cree una nueva carpeta, añadir un `README.md` que explique su propósito.

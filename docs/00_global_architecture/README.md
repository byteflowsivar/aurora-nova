# Arquitectura Global del Proyecto

Esta carpeta contiene las Decisiones de Arquitectura (ADRs) y los lineamientos técnicos que son **transversales a todo el proyecto Aurora Nova**.

Cualquier equipo que desarrolle un nuevo módulo debe adherirse a las decisiones documentadas en esta sección.

## Contenido

- **[ADR-001_Stack_Tecnologico_Principal.md](./ADR-001_Stack_Tecnologico_Principal.md)**: Define el stack de tecnologías base para todo el proyecto.
- **[ADR-002_Estandar_de_Claves_Primarias_UUIDv7.md](./ADR-002_Estandar_de_Claves_Primarias_UUIDv7.md)**: Establece el estándar para el diseño de claves en la base de datos.
